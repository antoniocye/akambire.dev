# How I Use Obsidian for Drafts

My writing pipeline is intentionally boring:

1. Start as quick notes in Obsidian.
2. Refactor into a clean markdown file.
3. Drop that file into this repo's blog folder.

## Current folder workflow

All blog posts live under `src/content/blog`.

Each markdown file can optionally include frontmatter:

```md
---
title: "My Post Title"
date: "2026-03-10"
---
```

The Finder view reads those values for display and sorting.
