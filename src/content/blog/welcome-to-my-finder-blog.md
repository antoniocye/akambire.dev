# Welcome to the Finder Blog

I wanted the blog section to feel like opening files in a desktop Finder window, not just scrolling another feed.

## Why this format?

- It mirrors how I already write notes in Obsidian.
- It keeps content simple: one `.md` file per post.
- It fits the style of this site, which already behaves like a mini operating system.

If you are reading this post in the app, markdown rendering is working end-to-end.
